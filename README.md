# 🍵 ตื่นมาก็ชาเย็น · WAKE UP CHA — เว็บไซต์โปรโมทร้าน

เว็บไซต์โปรโมทคาเฟ่ 4 สาขา + สินค้าส่งต่างจังหวัด + รีวิวลูกค้า
สไตล์โมเดิร์น มินิมอล โทนอบอุ่นแบบคาเฟ่ · รองรับมือถือเต็มรูปแบบ

---

## 📁 โครงสร้างไฟล์

```
fan-cafe-website/
├── index.html          ← โครงหน้าเว็บ
├── css/styles.css      ← ดีไซน์/สี (แก้ธีมสีได้ที่ส่วน :root บนสุด)
├── js/script.js        ← ★ ข้อมูลเมนู/สาขา/รีวิว/สินค้า อยู่ที่นี่ทั้งหมด
└── assets/images/      ← วางรูปภาพจริงที่นี่
```

---

## ✏️ วิธีแก้เนื้อหา (ง่ายมาก)

เปิดไฟล์ **`js/script.js`** แล้วแก้ลิสต์ด้านบนได้เลย:

- `MENU` — เมนูเครื่องดื่ม/ของอร่อย (ชื่อ, คำอธิบาย, ราคา)
- `BRANCHES` — ข้อมูล 4 สาขา (เวลาเปิด, ที่อยู่, ลิงก์ Google Maps)
- `SHIPPING` — สินค้าแพ็กส่งต่างจังหวัด
- `REVIEWS` — รีวิวจากลูกค้า

> 💡 แค่เปลี่ยนข้อความในเครื่องหมายคำพูด `"..."` ห้ามลบเครื่องหมาย `,` หรือ `{ }` นะคะ

---

## 🖼️ วิธีใส่รูปภาพจริง

1. นำรูปไปวางในโฟลเดอร์ `assets/images/`
2. ในไฟล์ `js/script.js` ใส่ชื่อไฟล์รูปในช่อง `img: ""` เช่น `img: "latte.jpg"`
3. รูปหลัก 2 จุดที่ตั้งชื่อไฟล์ไว้แล้ว — แค่วางรูปชื่อตรงกันก็ขึ้นเอง:
   - `assets/images/hero.jpg` → รูปแบนเนอร์หน้าแรก
   - `assets/images/about.jpg` → รูปในส่วน "เกี่ยวกับเรา"

> ถ้ายังไม่ใส่รูป เว็บจะแสดงกล่องสีสวยแทนชั่วคราว ไม่พังค่ะ

**ขนาดรูปแนะนำ:** แบนเนอร์ ~1920×1080px, เมนู/สินค้า ~800×800px

---

## 👀 วิธีดูเว็บบนเครื่อง

ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ได้เลย
หรือรันเซิร์ฟเวอร์ทดสอบ:

```bash
cd fan-cafe-website
python3 -m http.server 8000
# เปิด http://localhost:8000
```
